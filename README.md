Sebuah aplikasi *landing page* animasi premium yang dirancang dengan estetika *Dark Feminine* dan *Modern Tech-Core*. Proyek ini menampilkan integrasi video latar belakang tanpa interupsi, tata letak *Bento Box UI*, dan animasi transisi yang sangat mulus.

## ✨ Key Features
- **Cinematic Gatekeeper:** *Splash screen* interaktif untuk mengatasi limitasi *autoplay* audio pada *browser* modern.
- **Glassmorphism UI:** Penggunaan *backdrop-blur* dan *border* bernuansa *Rose Gold* untuk memberikan kesan tembus pandang yang mewah.
- **Bento Grid Layout:** Struktur data spesifikasi yang disajikan dalam asimetris *bento box* yang *trendy*.
- **Smooth Physics Animation:** Transisi halaman, *scroll reveal*, dan kustomisasi kursor menggunakan *spring physics* dari Framer Motion.

## 🛠️ Tech Stack
- **Framework:** React.js (Vite)
- **Styling:** Tailwind CSS v3
- **Animations:** Framer Motion
- **Routing:** React Router DOM
